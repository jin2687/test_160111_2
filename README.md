# 2D FLIP 流体シミュレーション

FLIP (Fluid-Implicit Particle) 法を使った2次元流体シミュレーションのWebアプリケーションです。

## 特徴

- **FLIP法**: パーティクルとグリッドを組み合わせた高精度な流体シミュレーション
- **リアルタイム描画**: Canvas APIを使用したスムーズなアニメーション
- **インタラクティブ**: マウスドラッグで流体に力を加えられます
- **パラメータ調整**: 重力、粘性、パーティクル数を調整可能

## 使い方

1. ブラウザでindex.htmlを開く
2. マウスをドラッグして流体を動かす
3. スライダーでパラメータを調整
